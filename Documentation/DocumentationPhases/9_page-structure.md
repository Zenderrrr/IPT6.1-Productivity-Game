# Page Structure
Wir legen in der Seitenstruktur fest, welche Seiten wir für die Website benötigen und bestimmen danach welche Funktion und Nutzen diese erfüllen. Das wird die Grundstruktur für die Entwicklung der Website sein, um genau zu wissen, was entwickelt werden muss und wozu es dient.

## Pages

### Homepage
**Zweck**:
- Einstiegsseite für neue Nutzer + Überblick über die App.

**Funktion**:
- Erklärung was die Website macht
- Einstieg ermöglichen
- Warum man die Website nutzen sollte (Motivation)

**Inhalt**:
- Titel / Hero Section
- Erklärung des Systems (Tasks -> XP -> Level -> Streak)
- Call-To-Action: Login / Registrierung

--- 

### Task Page
**Zweck**:
- Zentrale Arbeitsseite zum Verwalten und Erstellen von Funktionn

**Funktion**:
- Task erstellen / bearbeiten / löschen
- Tasks markieren als Erledigt
- Filtern und Übersichtsfunktionen

**Inhalt**:
- Task Liste (erledigt / offen)
- Button für neue Task erstellen
- Filter: z. B. Schwierigkeit, Kategorie, Status
- Task-Item: 
    - Titel
    - Schwierigkeit
    - Dauer
    - Status
- Aktionen:
    - Bearbeiten
    - Löschen
    - Erledigt

---

### Dashboard
**Zweck**:
- Man erhält eine Übersicht was man bisher gemacht hat, kann analysieren wie man sich bisher schlägt und erhält Motivation weiter zu machen.

**Funktion**:
- Aktuellen Fortschritt anzeigen
- Überblick über Aktivität geben

**Inhalt**:
- XP insgesamt
- Level + Fortschrittsbalken
- Streak
- Letzte erledigte Task
- Schneller Zugriff auf wichtige Funktionen
    - Task erstellen

---

### Login / Register
**Zweck**:
- Neue Benutzer können sich authentifizieren oder sich Anmelden, wenn sie schon einen Account haben.

**Funktion**:
- Login bestehender Benutzer
- Registrierung neuer Nutzer

**Inhalt**:
- Login:
    - Username / Email
    - Passwort
    - Button zum Einloggen

- Registrieren
    - Username
    - Email
    - Passwort
    - Button zum Registrieren

*Nach login weiterleiten zum Dashboard*

---

### Profil
**Zweck**:
- Persönliche Daten anzeigen und verwalten
**Funktion**:
- Benutzerinformationen anzeigen
- Daten bearbeiten
- Account verwalten

**Inhalt**:
- Benutzername
- Email
- Passwort ändern
- Account löschen / zurücksetzen
- Theme einstellungen (optional)
