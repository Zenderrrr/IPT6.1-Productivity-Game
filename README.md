# IPT6.1-Productivity-Game

## Projektziel und Projektbeschreibung

### Ziel
Beim Projekt handelt es sich um eine Website basierte anwendung, welche die Produktivität des Benutzers fördern soll. Dies wird mit einem Spielerischem Ansatz erreicht (Gamification). Die Website soll Nutzern ermöglichen Tasks zu verwalten und abzuarbeiten, gleichzeitig aber den Nutzern zu motivieren die Task zu erledigen. Dafür werden Elemente der Spielebranche benutzt wie XP, Level, Streaks, welche die Arbeit messbar macht und motivierend wirkt.

### Funktion (was kann der Nutzer?)
Der Nutzer kann Aufgaben erfassen, verwalten und als erledigt markieren. Beim Abschliessen von Aufgaben erhält der Nutzer Erfahrungspunkte (XP), steigt im Level auf und baut tägliche oder wöchentliche Streaks auf. Zusätzlich kann der Nutzer seine Produktivität anhand von Statistiken auswerten.

### Grobe Features
- Tasks verwalten: Aufgaben erstellen, löschen & aktualisieren
- XP & Level ups: XP sammeln, Level ups anzeigen
- Streaks: tägliche, wöchentliche Produktivitätsfolgen
- Statistiken: Auswertungen von erledigten Aufgaben nach Kriterien, angepasst an Benutzer
